NLP project

Detection of Stress in Social Media.
A staggering 70 million work days are lost each year due to mental health problems in the UK, costing employers approximately ¬£2.4 billion per year. This project called Dreaddit is a text corpus of lengthy multi-domain social media (Reddit) data for the identifaction of stress. The data set consists of 3.5k labelled segments by human annotators. Machine Learning and Deep Learning algorithms were tested.

  • Data Analytics with SQL, BigQuery, Pandas, Numpy, and Matplotlib
  • Statistics with Scipy, Seaborn and Statsmodels
  • Machine Learning with Scikit-learn; Deep Learning with TensorFlow Keras
  • Development of Data Products with Google Cloud Platform, ML Flow, Flask, Heroku and Streamlit
